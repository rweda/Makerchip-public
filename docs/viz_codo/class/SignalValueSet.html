Class: SignalValueSet

Defined in: ../doc/cleaned_coffee_public/SignalValueSet.coffee

Overview

Class representing a set of SignalValue's that can step signals in unison. This class has similar methods to those of SignalValue that operate on all signals in the set. Transition-based methods use a representative signal to identify transitions. Methods that can be chained in SignalValue can also be chained in SignalValueSet. The signals of the set are not required to have the same cycle value. Methods adjust signals in unison, preserving their relative cycle times. SignalValues of the set may be operated on outside of this class.
