Makerchip Plugin API

Overview

The Makerchip Plugin API enables third-party platforms to embed the Makerchip IDE in an iframe and interact with it programmatically. This allows you to integrate TL-Verilog circuit design and simulation capabilities into your own web applications.

This documentation details the classes and methods available for integration.

See also the Example Use Page

Getting Started

To embed Makerchip in your application:

  1. Include the plugin iframe:

    <iframe src="https://makerchip.com/plugin.html?iframeId=myCircuit"></iframe>

  2. Extend the IdePlugin class to customize behavior:

    window.defineIDE = function(IdePlugin) {
  return class MyPlatform extends IdePlugin {
    // Override methods to customize behavior
    async onReady() {
      // Called when IDE is ready
    }
  }
}

Multi-Instance Support

You can embed multiple independent IDE instances on a single page by using unique iframeId values and providing an object of initialization functions (which could be a common function):

window.defineIDE = {
  'ide1': function(IdePlugin) { /* ... */ },
  'ide2': function(IdePlugin) { /* ... */ }
}
